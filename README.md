# MiASMa

A game engine for the SEGA Mega Drive written in assembly for development on Windows.

## Installation

3 programs must be included your system's [PATH environment variable](https://en.wikipedia.org/wiki/PATH_(variable)).

* SN System's ASM68K version 2.53
* SN System's Psylink version 2.74
* [Devon Artmeier's mdromfix version 1.0](https://github.com/devon-artmeier/mdromfix/releases/tag/v1.0)

## Building

MiASMa can either be built by
* Run make
* (VS Code) Run Task/Build Task
